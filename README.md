## Developer
Requires:
* Node v10 or later.

### How to get started
* Fork repository, clone locally, navigate to repository directory,
* Download all the files with `npm i`,
* To test, run `npm test`. 

### How to use API
* Connect to server with `npm run start`.
* Enter `http://localhost:3000` in your browser.
